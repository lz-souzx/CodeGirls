CodeGirls — Projeto de Análise e Qualidade de Software

Descrição:
Documentação completa de um sistema desenvolvido para a clínica NutriLife, com foco em análise, requisitos, modelagem UML, prototipação e arquitetura de software. O projeto aborda desde a identificação do problema até a definição técnica do aplicativo que será utilizado por pacientes, nutricionistas, educadores físicos e funcionários administrativos.

Escopo do Sistema

O sistema proposto tem como objetivo digitalizar o atendimento da clínica, permitindo:

- Cadastro e gerenciamento de pacientes
- Planos alimentares e treinos personalizados
- Acompanhamento digital (fotos, IMC e notificações)
- Chat entre profissionais e usuários
- Controle administrativo e financeiro
- Construído com base em requisitos funcionais e não funcionais detalhados.

Principais Artefatos Incluídos:

- Levantamento de requisitos (funcionais e não funcionais)
- Mapeamento de usuários e permissões
- Diagrama de Caso de Uso
- Diagrama de Classes (UML)
- Modelagem de Banco de Dados (MySQL)
- Protótipos de telas para Web e Mobile
- Análise completa do problema e solução

Todos os artefatos estão reunidos no PDF do projeto.

Modelagem Técnica

- Linguagem proposta: Dart
- Framework: Flutter
- Banco de Dados: MySQL

Arquitetura utilizando:

- DTO
- DAO
- Model

- Estrutura de tabelas e relacionamentos normalizados

Destaques do Projeto:

- Documentação escrita profissionalmente
- Modelagem consistente e alinhada a boas práticas de engenharia de software
- Fluxos claros entre requisitos, diagramas e banco
- Protótipos funcionais que demonstram a experiência do usuário
- Projeto completo mesmo sem código-fonte, ideal para análise, QA e arquitetura

Status do Projeto: Concluido.
